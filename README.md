# Project Setup Instructions

⚠️ IMPORTANT — READ BEFORE STARTING

Before running this system, please follow the steps below carefully.

## Step 1: Vendor Folder Setup
This project requires the `vendor` folder to work properly.

You MUST:
1. Copy the `vendor` folder
2. Paste it into the following directories:
   - `/admin/vendor`
   - `/user/vendor`

⚠️ If the `vendor` folder is missing, the system will NOT run.

## Step 2: Database Configuration
- Open `config.php`
- Update database details (host, username, password, database name)

## Step 3: Run the Project
- Place the project inside your web server folder (e.g. `htdocs`)
- Open browser and access the project via `localhost`

## Notes
- Do not modify files inside the `vendor` folder
- The `vendor` folder contains required dependencies

✔️ Setup completed successfully
